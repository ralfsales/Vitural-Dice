🎲 Virtual Dice

A simple and interactive virtual dice application built with Java.
This project simulates the roll of a six-sided die, generating random results just like a real dice.
It was created as a practical exercise to strengthen object-oriented programming concepts, random number generation, and Java application development.

📋 How It Works:
When the user rolls the dice, the application generates a random number between 1 and 6 and displays the result.
Each roll is independent, ensuring a fair and unpredictable outcome.

🚀 Features
Roll a virtual six-sided dice
Random and realistic dice outcomes
Simple and user-friendly interface
Lightweight and easy to run
Great beginner-friendly Java project
🛠️ Technologies Used
Java
Object-Oriented Programming (OOP)
Java Random Number Generation
