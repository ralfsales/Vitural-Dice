🎲 Virtual Dice

A simple and interactive virtual dice application built with JavaScript.
This project simulates the roll of a six sides dice, generating random results just like a real dice.
It was created as a practical exercise to strengthen OOP - object oriented programming concepts, random number generation, and JavaScript application development.

📋 How It Works:
When the user rolls the dice, the application generates a random number between 1 and 6 and displays the result.
Each roll is independent, ensuring a fair and unpredictable outcome.

🚀 Features
Roll a virtual six sided dice
Random and realistic dice outcomes
Simple and user friendly interface
Lightweight and easy to run
Great beginner friendly JavaScript project

🛠️ Technologies Used
HTML | CSS | JavaScript
