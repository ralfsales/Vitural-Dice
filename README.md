# Virtual Dice

Static 3D virtual dice web app.

## Upload to GitHub

Upload every file in this folder to your repository:

- `index.html`
- `styles.css`
- `main.js`

For GitHub Pages, set the published page to use this folder as the site root, or place these files at the root of your repository. The page entry point is `index.html`.

No backend, database, login, server-side code, CDN, or external JavaScript library is required.
